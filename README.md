Project Title

Omnichannel Retail Sales Analysis Dashboard

Project Description

This project focuses on analyzing retail sales data using Python, SQL, and Power BI. It follows a structured four-week development approach, including data cleaning, database analysis, dashboard creation, and insight generation. The objective is to transform raw data into meaningful insights and support data-driven decision-making.

Tools & Technologies
-Python (Pandas) – Data Cleaning
-MySQL – Data Analysis
-Microsoft Power BI – Data Visualization

Week 1: Data Cleaning & Preparation
Performed data cleaning using Python (Pandas)
Removed null values and duplicate records
Standardized date formats for consistency
Validated data types across all columns
Files:
-sales_data.csv
-cleaned_data.csv
-data_cleaning.ipynb

Week 2: SQL Analysis
Steps Performed:
Created database: retail_db
Imported cleaned dataset into MySQL Workbench
Created and updated revenue column
Performed SQL queries for analysis:
-Total Revenue
=Total Quantity Sold
-Sales by Category
-Top Selling Products
-Sales Trends

Week 3: Dashboard Development
Built an interactive dashboard using Power BI
Created key visuals:
-Total Revenue (Card)
-Total Quantity Sold (Card)
-Sales by Category (Column Chart)
-Top Selling Products (Bar Chart)
-Sales Trend Over Time (Line Chart)
-Added slicer for category-based filtering
-Applied formatting, alignment, and consistent color theme

Week 4: Insights & Reporting
Key Insights:
-Electronics category generates the highest revenue
-Certain products show consistently high demand
-Sales trend indicates fluctuations across months
-Peak sales periods can be identified for better planning

Business Recommendations:
-Focus on high-performing categories for revenue growth
-Maintain inventory for top-selling products
-Plan marketing strategies during peak periods
-Improve performance of low-selling categories

Conclusion

This project demonstrates the complete data analytics workflow, from data preprocessing to visualization and insight generation. It highlights the importance of data-driven decision-making in retail business operations.
